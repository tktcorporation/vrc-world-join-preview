# VRC World Join Preview Generator

VRChatのワールドジョインリンク用のプレビュー画像を生成するWebアプリケーションです。

## 機能

- 複数のプレビューテンプレート
- カスタマイズ可能なデザイン
- ワンクリックで画像生成
- モダンなUI/UX

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# コードのフォーマット
npm run format
```

## 技術スタック

- React 18
- TypeScript
- Vite
- TailwindCSS
- Biome (Linter/Formatter)
- html2canvas
